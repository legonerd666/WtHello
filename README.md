## Dependencies:

    -Wt: https://redmine.webtoolkit.eu/projects/wt/wiki/Wt_Installation
    -Cmake: 3.7
    -Boost Library (for wt)

## How to build and run this program on linux:

    1. Clone Repo
    2. Download Dependencies
    3. Open terminal in the cloned directory
    4. mkdir build
    5. cd build
    6. cmake ..
    7. make
    8. ./hello --docroot . --http-address 0.0.0.0 --http-port 9090
    9. Open Browser and put in http://127.0.0.1:9090/

## What this program does:

    This is just a simple hello world program using Wt.

## File Structure:

    All the relevant code is in the src and include directories
    src/:
        HelloApplication.cpp
        main.cpp

    include/:
        HelloApplication.h

## Class Structure:

    HelloApplication (extends WApplication):
        WLineEdit *nameEdit_
        WText *greeting_

## Documentation:

    You can get the full documentation by following these steps:
        1. Clone the repo
        2. Install Doxygen and its dependencies (Flex and Bison)
        3. In the terminal run "doxygen doxygenConfig"
        4. Open the html folder generated by doxygen
        5. Open index.html in your browser
